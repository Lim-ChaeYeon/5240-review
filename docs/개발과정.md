## 🎯 프로젝트 개요

### 목표
Microsoft Edge 2025 연도 결산 페이지를 벤치마킹하여 팀 연도 결산을 보여주는 인터랙티브 웹 페이지 제작

### 요구사항
1. **Microsoft Edge 스타일 재현**
   - 수직 슬라이드 전환
   - 부드러운 애니메이션
   - 월별 성과 표시

2. **추가 기능 요청**
   - 화면 클릭으로 다음 슬라이드 이동
   - 성과물을 대표하는 이미지들이 둥둥 떠다니는 효과
   - 인트로 화면의 이미지들이 마우스 움직임에 반응
   - 월별 성과물을 보여줄 때 대표 이미지 배치
   - 동영상 자동 재생
   - 마지막 페이지에서 클릭으로만 엔딩 화면 이동

## 🔍 기술 스택 분석

### Microsoft Edge 실제 사용 기술 (분석 결과)

**프레임워크**
- Nuxt.js 3 (Vue.js 기반 SSR)
- Vite (빌드 도구)

**커스텀 컴포넌트**
- `PxSlides` - 슬라이드 시스템
- `SlidePlayer` - 플레이어 컨트롤
- `ParallaxScene` - 패럴랙스 효과
- `data-reveal` 디렉티브 - 커스텀 애니메이션

**특징**
- 외부 라이브러리(Swiper, GSAP) 사용하지 않음
- 완전 커스텀 솔루션
- Azure Front Door CDN 사용

### 우리 프로젝트 선택 기술 스택

**접근 방식**: 빠른 구현을 위해 검증된 라이브러리 사용

| 기술 | 버전 | 용도 |
|------|------|------|
| **Swiper.js** | 11.x | 슬라이드 효과 및 전환 |
| **AOS** | 2.3.x | 스크롤 애니메이션 |
| **CountUp.js** | 2.8.x | 숫자 카운트업 애니메이션 |
| **바닐라 JS** | ES6+ | 커스텀 로직 구현 |
| **CSS3** | - | 애니메이션 및 스타일링 |

```javascript
// 구현 방법
swiperContainer.addEventListener('click', function(e) {
```

#### 요청사항 3: 마우스 반응 이미지 (인트로)

```javascript
// 패럴랙스 효과 구현
마우스 위치 감지 (mousemove)
    ↓
각 이미지의 깊이(depth)에 따라 이동 거리 계산
    ↓
requestAnimationFrame으로 부드러운 애니메이션
```

**구현 상세**
- 12개 이미지 배치
- 각각 다른 depth 값 (0.1, 0.2, 0.3)
- 마우스 움직임에 따라 -50px ~ +50px 이동
- 랜덤 회전 효과 (-10° ~ +10°)

#### 요청사항 4: 월별 대표 이미지 배치

**위치 패턴 시스템** (8가지)
```javascript
positions = [
    'top-left',      // 좌측 상단 (-5도)
    'top-right',     // 우측 상단 (+5도)
    'bottom-left',   // 좌측 하단 (+3도)
    'bottom-right',  // 우측 하단 (-3도)
    'left-center',   // 좌측 중앙 (-8도)
    'right-center',  // 우측 중앙 (+8도)
    'top-center',    // 상단 중앙 (-2도)
    'bottom-center'  // 하단 중앙 (+2도)
];
```

**자동 배치 알고리즘**
```javascript
positionIndex = (slideIndex * 3 + mediaIndex) % 8
// 슬라이드마다 다른 위치 패턴
```

**스타일**
- 둥근 모서리 (20px)
- 강한 그림자 효과
- 반투명 테두리
- 호버 시 1.05배 확대

#### 요청사항 5: 동영상 자동 재생

```html

```

**설정**
- `autoplay` - 자동 재생

### Phase 3: UX 개선 (v2.1.0)

#### 요청사항 6: 마지막 슬라이드에서 클릭으로만 완료

**기존 문제**
```
마지막 슬라이드 도달
    ↓
1.5초 자동 대기
    ↓
자동으로 엔딩 화면 ❌ (사용자가 못 볼 수도 있음)
```

**개선 사항**
```
마지막 슬라이드 도달
    ↓
"🎉 클릭하여 결산 완료" 힌트 표시
    ↓
사용자 클릭 대기 ⏳
    ↓
클릭 시 엔딩 화면 ✅
```

**구현 방법**
```javascript
// reachEnd 이벤트 수정
reachEnd: function() {
    // setTimeout 제거
    checkIfLastSlide(teamYearData.length - 1);
}

// 클릭 이벤트에서 처리
if (isLastSlide) {
    showEndingScreen();  // 마지막에만 엔딩
} else {
    swiper.slideNext();  // 나머지는 다음으로
}
```

**힌트 디자인**
- 위치: 화면 중앙 하단
- 스타일: 그라데이션 배경 + 강한 그림자
- 애니메이션: 펄스 (1.0 ↔ 1.1 scale)
- 아이콘: 좌우 흔들림 (-10° ↔ +10°)

## ✨ 주요 기능

### 1. 인터랙티브 인트로 화면
- 마우스 패럴랙스 이미지
- 부드러운 페이드 인 애니메이션
- "시작하기" 버튼

### 2. 월별 슬라이드
- 수직 페이드 전환
- 1-3개 이미지 자동 배치
- 동영상 자동 재생 (선택)
- 숫자 카운트업 애니메이션
- 세부 항목 표시

### 3. 떠다니는 배경 효과
- 15개 랜덤 아이콘
- 4가지 애니메이션 패턴
- GPU 가속 (will-change)

### 4. 다양한 조작 방법
- **마우스 휠** - 슬라이드 이동
- **화면 클릭** - 다음 슬라이드
- **Space/화살표** - 키보드 네비게이션
- **스와이프** - 모바일 터치
- **버튼** - 네비게이션 버튼

### 5. 사용자 피드백
- 진행률 바 (상단)
- 슬라이드 카운터 (우측 상단)
- 클릭 힌트 (우측 하단)
- 완료 힌트 (중앙 하단, 마지막만)

### 6. 완벽한 반응형
- 데스크톱 (1920px+)
- 태블릿 (768px-1024px)
- 모바일 (320px-767px)
- 이미지/동영상 크기 자동 조정

### 기술적 인사이트

1. **라이브러리 선택의 중요성**

4. **사용자 경험**
