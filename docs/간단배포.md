```

```

---

## 📋 3단계로 끝내기

### 1️⃣ GitHub 저장소 만들기 (1분)

1. https://github.com 접속 → 로그인
2. 우측 상단 **"+"** → **"New repository"**
3. 입력:
   ```
   Repository name: review
   Public ✅ (중요! 체크하세요)
   ```
4. **"Create repository"** 클릭

✅ 완료! 저장소가 생성되었습니다.

---

### 2️⃣ 파일 업로드하기 (2분)

**방법 A: 자동 스크립트 사용** ⭐ 가장 쉬움!

```bash
cd /home/sdh/5240/review
./deploy.sh
```

그냥 엔터/엔터 누르면 끝!

---

**방법 B: 터미널에서 직접**

```bash
cd /home/sdh/5240/review

# 초기화
git init
git add .
git commit -m "팀 결산 페이지"

# GitHub 연결 (YOUR_USERNAME을 본인 것으로!)
git remote add origin https://github.com/YOUR_USERNAME/review.git
git branch -M main
git push -u origin main
```

---

**방법 C: 웹에서 직접 업로드**

1. GitHub 저장소 페이지에서
2. **"uploading an existing file"** 클릭
3. 파일 **전체 선택** → 드래그 앤 드롭
4. **"Commit changes"** 클릭

✅ 완료! 파일이 업로드되었습니다.

---

### 3️⃣ GitHub Pages 켜기 (30초)

1. GitHub 저장소에서 **"Settings"** 탭
2. 왼쪽 메뉴 → **"Pages"**
3. Source 설정:
   ```
   Branch: main
   Folder: / (root)
   ```
4. **"Save"** 클릭
5. 1-2분 대기...
6. 완료! 🎉

```
✅ Your site is live at
https://YOUR_USERNAME.github.io/review/
```

## 💡 더 쉬운 방법: Netlify Drop

**GitHub 계정도 필요 없어요!**

1. https://app.netlify.com/drop 접속
2. `/review` 폴더를 **드래그 앤 드롭**
3. 완료!

```
https://random-name-123.netlify.app
```

**장점**: 초간단 (1분 완성)  
**단점**: 수정할 때마다 다시 업로드 필요
